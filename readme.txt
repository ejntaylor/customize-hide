== Usage ==

Just activate and then press Control and Z in the admin backend.

== Installation ==

Installing "WordPress Customize Hide” can be done either by searching for “Customize Hidel" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org.
1. Upload the ZIP file through the "Plugins > Add New > Upload" screen in your WordPress dashboard.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Visit the settings screen and configure, as desired.
